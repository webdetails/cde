NewMapComponent

Description

This component allow the user to navigate through the map and see information about marked locations. This locations are obtained with a Data Source that returns a result set where certain column names have associated semantics:

Locations Result Set

Longitude
Location Longitude
Latitude
Location Latitude
Address
Location address. Either one of the columns or Latitude/Longitude must be present
City
Location address. Either one of the columns or Latitude/Longitude must be present
County
Location address. Either one of the columns or Latitude/Longitude must be present
State
Location address. Either one of the columns or Latitude/Longitude must be present
Region
Location address. Either one of the columns or Latitude/Longitude must be present
Country
Location address. Either one of the columns or Latitude/Longitude must be present
Description
Optional. If existent, its value will be used to populate an hover popup for each location
marker
Optional. If exists, this will be used as the url for the marker image to be used
markerWidth
Optional. Width of the marker image
markerHeight
Optional. Height of the marker image
popupContents
Optional. If exists, the value will be used as the html to populate a popup window that will appear when user clicks on the marker.
popupWidth
Optional. Width for the popup window
popupHeight
Optional. Height for the popup window.

Options

listeners
Array - Parameters who this component will react to
parameters
Array of Arrays - Parameters
htmlObject
map. Not optional for now
executeAtStart
True to execute the component at start, false otherwise
preExecution
Function - Function to be called before the component is executed
postExecution
Function - Function to be called after the component is executed
Center Longitude
Initial longitude position
Center Latitude
Initial latitude position
Default zoom Level
Initial map zoom
Datasource
Name for the cda data source where the location data is read from
Map Engine
Map engine to use to render the map. Can be either google (Google Maps) or open (OpenStreetMaps). If using Google Maps, please define a global javascript variable with the name API_KEY containing the string with your API key.
Marker image
URL-string for the marker image. This will be override either by markers defined in the resultset or when using cgg graphs to render markers.
Marker Width
Width for the marker image.
Marker Height
Height for the marker image.
Marker Click Parameters
Parameters to be changed when a marker is clicked - arg is column Name, value is parameter name to change with column value
Marker click Function
Javascript function to be executed when marker is clicked
Cgg Graph for Marker
Name of the cgg graph to use as marker renderer. This will usually be in the form /solution/GraphName.cdw
Cgg Graph Parameters
Mapping between resultset columns and parameters for cgg graph - arg is column Name, value is the parameter name that will be passed to CGG having as value the column value
Div for popup window
Div id to use as the source for the popup window contents
Popup Width
Width for the popup window.
Popup Height
Height for the popup window.
